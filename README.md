Coding Resources Overflow
=========================
A web-app for Dev Bootcamp alumni to share and vote on programming learning resources.

A WIP by [Kenneth Yu](https://github.com/cpkenn09y) and [Dan Bender](https://github.com/danbender).

To download the repo:
1) open Terminal (Mac OSX app)
2) cd Desktop
3) git clone https://github.com/danbender/coding_resources_overflow.git
4) cd coding_resources_overflow

And then, to run locally:
1) bundle
2) bundle exec rake db:create
3) bundle exec rake db:migrate
4) bundle exec rake db:test:prepare
5) bundle exec rspec
6) rails s
